# Multi-purpose Dockerfiles

Multi-purpose Dockerfiles (wrapper, dummy data, etc)

## Wrapper images

1. `fluentd-es-sidecar`: Wrapper image of the Fluentd to use as a Elasticsearch client sidecar
2. `mongodb`: Wrapper image of the MongoDB to initiate database

## Dummy generator images

1. `py-timelog`: Writes the log of the current time to the given file path.
